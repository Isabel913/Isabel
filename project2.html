Project 2

Control

System Overview

In this Project, I connected a Photon to a breadboard, and uses Potentiometers and buttons to control the display of a pattern which I coded with processing.

Board View

Particle.io View

Potentiometer and Button

Potentiometer Changes Pattern Size

When the Potentiometer on the board is turned clock wise, the size of the pattern will get bigger, when the Potentiometer is turned counter-clock wise, the pattern will get smaller.

Button Changes Display Color

As the button on the board is pressed, the Hue, Saturation, and Brightness of the rotating circle will change randomly
